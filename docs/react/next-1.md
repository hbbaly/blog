## 安装

```bash
npm init next-app next-app --example "https://github.com/vercel/next-learn-starter/tree/master/learn-starter"
```

安装好之后

```bash
cd next-app
```

查看package.json

```json
// scripts命令
"scripts": {
  "dev": "next dev",
  "build": "next build",
  "start": "next start"
}
```

```bash
npm run dev
```

打开 `localhost:3000` 可以看到
