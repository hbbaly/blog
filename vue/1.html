            【vue】$attrs 与 $listeners

            hbbaly 2018-12-28 Vue

            主要解决组件嵌套传参问题，在高阶组件特别有用

            # $attrs

            attrs 包含了父作用域中不作为 prop 被识别 (且获取) 的 attribute 绑定 (class 和 style 除外)。当一个组件没有声明任何 prop 时，这里会包含所有父作用域的绑定 (class 和 style 除外)，并且可以通过v-bind="$attrs" 传入内部组件——在创建高级别的组件时非常有用

            # $listener

            包含了父作用域中的 (不含 .native 修饰器的) v-on 事件监听器。它可以通过 v-on="$listeners" 传入内部组件——在创建更高层次的组件时非常有用。

            # 示例

            attrs.vue文件




             




             















            <template>
  <div class='example-attrs'>
    示例
    <attrs-child @action="actionFirst" @actionSecond="actionSecond" name="hbb" age='20' sex='1'  />
  </div>
</template>

<script>
  import AttrsChild from './attrsChild'
  export default {
    components: {
      AttrsChild
    },
    methods: {
      actionSecond () {
        console.log('actionSecond');
      },
      actionFirst () {
        console.log('actionFirst');
      }
    },
  }
</script>

            上述代码可以得知： 向AttrsChild 组件传递name, age, sex参数及actionFirst,actionSecond函数

            AttrsChild.vue



             




             


























            <template>
  <div class='attrs-son'>
    <attrs-grand v-bind="$props" v-on="$listeners" />
  </div>
</template>

<script>
  import AttrsGrand from './attrsGrand'
  export default {
    components: {
      AttrsGrand
    },
    props: {
      name: {
        type: String,
        default: ''
      },
      age: {
        type: Number,
        default: 0
      },
      sex: {
        type: Number,
        default: 0
      }
    },
    mounted() {
      console.log(this.$attrs, 'attrs', this.$props, this.$listeners);
      this.$listeners.action()
      this.$listeners.actionSecond()
    },
  }
</script>

            上述代码可知：

            1. attrsChild组件引用了AttrsGrand组件。

            2. 第三行代码，向 AttrsGrand组件传递了$attrs, $listeners。

            attrsGrand.vue



             













            <template>
  <div class='grand-son'>
    {{$attrs.name}}--{{$attrs.age}} -- {{$attrs.sex}}
  </div>
</template>

<script>
  export default {
    mounted() {
      console.log(this.$attrs, '====123==', this.$props);
      this.$listeners.action()
      this.$listeners.actionSecond()
    },
  }
</script>

            # provide, inject

            provide 和 inject 主要在开发高阶插件/组件库时使用。并不推荐用于普通应用程序代码中。

            这对选项需要一起使用，以允许一个祖先组件向其所有子孙后代注入一个依赖，不论组件层次有多深，并在其上下游关系成立的时间里始终生效。

            provide 选项应该是一个对象或返回一个对象的函数。 inject 选项应该是一个字符串数组，或一个对象，对象的 key 是本地的绑定名，value 是在可用的注入内容中搜索用的 key (字符串或 Symbol)，或 一个对象。

            // parent
export default {
  provide: {
    foo: 'bar'
  },
  // ......
}

            // son
export default {
  inject:['foo']
  // ......
}

            1. AttrsGrand组件中使用attrs中传递过来的参数

            2. AttrsGrand的mounted中使用$listeners触发attrs中的函数

            3. vuex, $attrs, $listener, provide与inject也是高阶组件传递参数常用的放法。
