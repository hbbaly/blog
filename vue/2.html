        【vue】监听生命周期

        hbbaly 2019-01-02 Vue

        # 监听内部生命周期

        某个页面需要在created中设置了一个定时器， 通常我们会这样写：

        data () {
  return {
    timer: ''
  };
},
created () {
  this.timer = setInterval(() => {
    console.log('timer');
  }, 1000);
},
destroyed() {
  clearInterval(this.timer)
}

        使用hook钩子我们只需：

        created () {
  this.timer = setInterval(() => {
    console.log('timer');
  }, 1000);
  this.$once('hook:destroyed', () => {
    clearInterval(this.timer)
  })
},

        在Vue组件中，可以用过$on,$once去监听所有的生命周期钩子函数，如监听组件destroyed的钩子函数可以写成 this.$on('hook:destroyed', () => {})

        # 监听外部生命周期

        如果用了一个第三方组件，需要监听第三方组件数据的变化，但是组件又没有提供相应的事件，可以在外部监听组件的相应的钩子函数。

        <watch-child @hook:mounted = 'childMounted' @hook:updated="childUpdate" />

        使用@hook:mounted, @hook:updated去监听自组件的mounted执行完毕，以及自组件的数据更新。

        # $watch 用法












         

         






         






        data () {
  return {
    timer: '',
    testData: {
      num: 0
    }
  };
},
created () {
  this.timer = setInterval(() => {
    console.log('timer');
    this.testData.num += 4
  }, 1000);
  const watchTime = this.$watch('testData', () => {
    console.log(this.testData.num, 'testData发生了变化');
  },{
    deep: true,
    immediate: true
  })
  setTimeout(() => {
    watchTime()
  }, 3000);
  this.$watch('hook:destroyed', () => {
    clearInterval(this.timer)
  })
}

        上述代码可知： 第14行通过this.$watch监听testData对像改变, 14行在3s后结束监听
