字符串的扩展
发表于 2017-04-05| 更新于 2019-10-15|ES6

includes

返回布尔值，表示是否找到了参数字符串。

startsWith

返回布尔值，表示参数字符串是否在原字符串的头部。

endsWith

返回布尔值，表示参数字符串是否在原字符串的尾部。

repeat

方法返回一个新字符串，表示将原字符串重复n次

padStart(),padEnd()

如果某个字符串不够指定长度，会在头部或尾部补全。padStart()用于头部补全，padEnd()用于尾部补全。

字符串模板

用反引号（`）标识。
